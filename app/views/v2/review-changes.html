How will you agree changes?

There will be times when plans will need to change, such as if one parent is too unwell to care for children.

Think about:

  • how changes should be agreed. For example, by email.
  • how much notice you must give the other parent

Heather has suggested:

Let’s give each other at least 48 hours notice if plans need to change. We should agree changes by text message.
